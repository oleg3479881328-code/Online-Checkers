# Online Checkers ♟

**Русские шашки** — многопользовательская онлайн-игра с поддержкой WebSocket.

## Возможности

- 🎮 Игра по сети в реальном времени
- 🏠 Создание комнаты и приглашение друзей по ID
- 👑 Поддержка дамок (шашки становятся дамками при достижении последнего ряда)
- ⚡ Обязательное взятие (если есть возможность побить — нужно бить)
- 🔄 Цепочка взятий (можно бить несколько шашек за ход)
- 🎨 Современный адаптивный дизайн
- 📋 Копирование ID комнаты в буфер обмена

## Как играть

1. **Создайте комнату** — нажмите "Создать комнату"
2. **Поделитесь ID** — отправьте другу 6-символьный ID комнаты
3. **Присоединитесь** — друг вводит ID и нажимает "Присоединиться"
4. **Играйте!** — белые ходят первыми

## Правила

- Игра ведётся на доске 8×8
- Белые ходят первыми
- Шашки ходят по диагонали на одну клетку вперёд
- Если шашка достигает последнего ряда, она становится дамкой
- Дамка может ходить по диагонали в любом направлении
- Взятие обязательно — если есть возможность побить шашку соперника, вы обязаны это сделать
- После взятия можно продолжить бить той же шашкой (цепочка)

## Установка и запуск

```bash
# Клонировать репозиторий
git clone https://github.com/oleg3479881328-code/Online-Checkers.git
cd Online-Checkers

# Установить зависимости
npm install

# Запустить сервер
npm start
```

Сервер будет доступен по адресу: `http://localhost:3000`

## Технологии

- **Backend:** Node.js, Express, Socket.IO
- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Протокол:** WebSocket (Socket.IO)
